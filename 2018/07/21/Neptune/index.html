数据储存方案

文件存储：

创建对象：
openFileOutput(文件名，操作模式)方法获得FileOutPutStream对象
FileOutPutStream类的对象通过调用OutputStreamWriter(xx)方法将字节流对象转换成字符流对象
最后通过BufferedWriter(xx)调用字符流对象writer实例化BufferedWriter对象

便可以进行储存操作：

BufferedWriter writer = new BufferedWriter(new OutputStreamWriter(openFileOutput(文件名，
操作模式)))；
writer.write(xx数据)；

读取操作同理：

BufferedReader reader = new BufferedReader(new InputStreamReader(openFileInput(文件名，
操作模式)));
while((str = reader.readLine()!=null)  { StringBuilder.append(str)}

核心技术：openFileInput()和openFileOutput()
